# AI Chat Timestamp

AI 채팅 사이트에서 메시지를 보낼 때 자동으로 타임스탬프를 추가해주는 크롬 확장 프로그램

## 지원 사이트

- Claude.ai
- ChatGPT
- Gemini

## 설치 방법

**[📦 ZIP 다운로드](https://github.com/kansuki20/ai-chat-timestamp/raw/refs/heads/main/ai-chat-timestamp.zip)**

1. ZIP 파일 압축 해제
2. 크롬 주소창에 `chrome://extensions` 입력
3. 우측 상단 **개발자 모드** 활성화
4. **압축해제된 확장프로그램 로드** 클릭
5. 압축 풀린 폴더 선택
6. 크롬 완전히 종료 후 재시작 (탭만 닫으면 안 됨)

## 사용법

확장 프로그램 아이콘 클릭 → 토글로 ON / OFF
